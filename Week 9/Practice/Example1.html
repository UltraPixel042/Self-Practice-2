การทดสอบ appendChild()

appendChild() คืออะไร?

appendChild() เป็นเมธอดใน JavaScript ที่ใช้สำหรับเพิ่มโหนด (node) ลงในท้ายสุดของรายการลูก (children) ของโหนด parent ที่กำหนด

// รูปแบบการใช้งาน
parentElement.appendChild(childElement);

รายการปลายทาง

คุณสมบัติสำคัญของ appendChild():

  • เพิ่มโหนดลงในตำแหน่งท้ายสุดของ parent
  • หากโหนดที่เพิ่มมี parent อยู่แล้ว จะถูกย้ายจากตำแหน่งเดิม
  • ส่งคืนโหนดที่เพิ่มเข้าไป
  • รองรับการเพิ่มได้ทั้ง Element, Text, Comment nodes

ตัวอย่างโค้ด:

// สร้าง element ใหม่
const newElement = document.createElement('div');
newElement.textContent = 'รายการใหม่';

// เพิ่มเข้าไปใน parent
document.getElementById('targetBox').appendChild(newElement);
